## debug(message, ...additionalMessages)

**Type:** Function  
**Arguments:**
  - message  
    **Type:** any  
    **Required:** Yes  
    **Description:** A message to log.
  - ...additionalMessages  
    **Type:** Array&lt;any&gt;  
    **Required:** No  
    **Description:** Additional messages to log.

**Returns:** void  
**Description:** Logs a message at debug level.

```ts
log.debug('Log this message!');
```
