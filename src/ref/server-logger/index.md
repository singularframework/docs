# ServerLogger

A class for logging information on console and disk, available [globally](../globals/#log) as `log`. The following methods are available on the `log` global:
  - [debug(message, ...additionalMessages)](#debugmessage-additionalmessages)
  - [info(message, ...additionalMessages)](#infomessage-additionalmessages)
  - [notice(message, ...additionalMessages)](#noticemessage-additionalmessages)
  - [warn(message, ...additionalMessages)](#warnmessage-additionalmessages)
  - [error(message, ...additionalMessages)](#errormessage-additionalmessages)
  - [id(sessionId)](#idsessionid)
